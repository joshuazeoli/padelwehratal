PRODUKTBILDER FÜR DEN CUBE-SHOP
================================

Lege hier deine Produktfotos ab – sie erscheinen automatisch im Shop
(statt der gezeichneten Icons). Erlaubte Formate: .jpg, .png oder .webp
Empfohlen: quadratisch oder 4:3, mindestens 800 px breit.

WICHTIG: Der Dateiname muss exakt der Artikel-Nummer entsprechen:

  shirt-cube-weiss.jpg              → Exercise Shirt Cube (Weiß)
  shirt-cube-schwarz.jpg            → Exercise Shirt Cube (Schwarz)
  shirt-cube-blau.jpg               → Exercise Shirt Cube (Blau)
  short-cube-weiss.jpg              → Exercise Short Cube (Weiß)
  short-cube-schwarz.jpg            → Exercise Short Cube (Schwarz)
  racket-viper-lebron-31.jpg        → Viper Juan Lebron 3.1
  racket-viper-soft-lebron-31.jpg   → Viper Soft Juan Lebron 3.1
  racket-veron-lebron-31.jpg        → Veron Juan Lebron 3.1
  racket-technical-viper-31.jpg     → Technical Viper 3.1
  racket-technical-viper-soft-31.jpg→ Technical Viper Soft 3.1
  racket-counter-viper-30.jpg       → Counter Viper 3.0
  racket-counter-viper-soft-30.jpg  → Counter Viper Soft 3.0
  racket-air-viper-27.jpg           → Air Viper 2.7
  racket-air-veron-27.jpg           → Air Veron 2.7
  balls-court.jpg                   → Court Bälle – 3er Pack
  balls-ace.jpg                     → ACE Bälle – 3er Pack

(statt .jpg geht auch .png oder .webp)

NICHT VERGESSEN: Die Bilder auch in den Ordner
github-deploy/produktbilder/ kopieren und zu GitHub hochladen,
damit sie auf der Live-Seite erscheinen.
